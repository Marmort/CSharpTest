1. Download MadCow Source.
2. Uncompress over a short path folder (e.g C:\MadCow\).
3. Open MadCow project solution file MadCow2011.sln.
4. Run/Compile.
5. MadCow will be placed over <Folder you extracted MadCow>\bin\Debug.
6. Enjoy.
